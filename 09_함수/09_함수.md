# 09_함수 (Function) ⚒️

- 특정한 기능을 하는 코드의 조각 (묶음)
- 특정 코드를 필요시에만 호출하여 간편하게 사용



- 분해 : 기능을 분해하고 재사용 가능
- 추상화 : 재사용성, 가독성, 생산성



- 함수의 종류

  1. 내장 함수
     - 파이썬에 기본적으로 포함된 함수

  2. 외장 함수
     - import 문을 통해 사용, 외부 라이브러리에서 제공
  3. 사용자 정의 함수
     - 직접 사용자가 만드는 함수
