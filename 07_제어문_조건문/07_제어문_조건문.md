# 07. 제어문_조건문 (Control Statement)

- 참/거짓을 판단할 수 있는 조건식과 함께 사용



- 기본형식

  - 조건에는 참/거짓에 대한 조건식

    ```python
    if condition == True:
    	# Code
    else:
        # Code
        
    # 들여쓰기 주위 (PEP8에서는 4space를 권장)
    ```

    ​     

  - 조건문 예시

    ```python
    a = 5
    if a > 4:
        print('4 초과')
    else:
        print('4 이하')
    
    # 4 초과
    ```

    ```python
    a = 2
    if a > 4:
        print('4 초과')
    else:
        print('4 이하')
    
    # 4 이하
    ```

  

   -  조건문 실습

         num = int(input('숫자 : '))
